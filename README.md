# Rime词库增强

按照官网给出的定义：Rime不是一种输入法。是从各种常见键盘输入法中提炼出来的抽象的输入算法框架。英文Rime涵盖了大多数输入法的共性，所以在不同设定下，Rime可化身为不同的输入法用来打字。Rime是跨平台的输入法软件，Rime输入方案可通用于以下发行版：

- 【中州韻】 ibus-rime → Linux
- 【小狼毫】 Weasel → Windows
- 【鼠鬚管】 Squirrel → Mac OS X
这个库收集了许多rime词库，具体用法见[官网](http://rime.im/)
